# ChatKit Customer Support Demo

An AI-powered airline customer support demo built with OpenAI ChatKit.

## Features

- Live itinerary data with real-time updates
- Customer profile management
- Service timeline tracking
- Meal preference selection via interactive widgets
- Streaming AI responses

## Running on Replit

1. Import this repo on Replit
2. Add your `OPENAI_API_KEY` in Secrets
3. Click Run

## Local Development

```bash
# Set your API key
export OPENAI_API_KEY="your-key"

# Run the app
bash run.sh
```
